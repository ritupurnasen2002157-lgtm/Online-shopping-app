# 🛍️ Fashion in Trend – E-commerce Website

## 📌 Project Overview

**Fashion in Trend** is a fully responsive front-end e-commerce website developed using **HTML, CSS, and JavaScript**.
The project provides a modern shopping experience with features like product browsing, cart management, wishlist, and user authentication.

---

## 🚀 Features

### 🛒 Product Management

* Dynamic product listing using JavaScript
* Category-wise filtering (Men, Women, Kids, etc.)
* Search functionality for products
* Product details modal with size selection

### ❤️ Wishlist System

* Add/remove products to wishlist
* Stored using localStorage
* Wishlist count updates dynamically

### 🛍️ Shopping Cart

* Add to cart functionality
* Update product quantity
* Remove items from cart
* Price calculation:

  * Subtotal
  * Discount (10%)
  * Tax (18%)
  * Shipping charges
* Checkout system with order summary

### 👤 User Authentication

* User registration and login system
* Password validation and strength checker
* Session handling using localStorage/sessionStorage
* Demo user support

### 🎨 UI/UX Features

* Responsive design (mobile + desktop)
* Interactive product cards with hover effects
* Hero image slider
* Modal popups (cart, wishlist, product details)
* Notification system for user actions

---

## 🧠 Code Structure

### 📄 index.html

* Main structure of the website
* Includes:

  * Hero section
  * Categories
  * Product grid
  * Modals (Cart, Wishlist, Login, Product Details) 

### 🎨 styles.css

* Handles styling and layout
* Responsive design
* UI components like cards, buttons, modals

### ⚙️ script.js

* Core functionality of the application
* Includes:

  * Product database 
  * Cart & wishlist logic 
  * Filtering & search system 
  * Authentication system 
  * Event handling and UI updates

---

## 💾 Data Storage

* Uses **localStorage** to store:

  * Cart items
  * Wishlist items
  * User data
* Ensures data persistence even after page refresh

---

## ⚙️ How to Run the Project

1. Download or clone the project
2. Open `index.html` in any browser
3. Start exploring the website

---

## 📊 Functional Flow

1. User visits homepage
2. Browses products or filters categories
3. Adds items to cart/wishlist
4. Registers or logs in
5. Proceeds to checkout
